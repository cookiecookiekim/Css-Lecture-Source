기본 속성 선택자

기본 선택자 뒤에 [] 기호를 사용하여 속석과 속성 값을 사용할 수 있다. 주로 id나 클래스 속성이 아닌 것들에 적용
선택자[속성명=속성값]{설정할 내용}
